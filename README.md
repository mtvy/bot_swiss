# bot_swiss

## Description

This is a telegram bot for swiss_lab

## Getting started

**First of all, clone this repository**

`git clone https://github.com/mtvy/bot_swiss.git /your/local/directory/`

---

**Go to project's directory and make bot's configs**

`make configs`

**To start bot**

`make start_bot`

**To stop bot**

`make stop_bot`

**To restart bot**

`make restart_bot`

**To clean all**

`make clean`
